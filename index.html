System Architecture

The system follows a modular architecture with four main components, each representing a real-world concept in robotics. This design allows for easy extension, testing, and maintenance.

Grid Class

Represents the environment with obstacles and boundaries. Handles coordinate validation and display.

Robot Class

The autonomous agent with position, direction, and movement capabilities. Implements navigation logic.

Sensor Class

Simulates obstacle detection. Checks if a position contains an obstacle before the robot moves.

Controller Class

The decision-making unit. Combines sensor input with robot state to make navigation decisions.

Python 3.x Pygame Object-Oriented Design Simulation Algorithm Design

OOP Principles Implemented

Encapsulation

Each class encapsulates its own data and methods. The Robot's position and direction are private attributes accessed only through controlled methods.

Abstraction

Complex navigation logic is hidden behind simple interfaces. Users call `controller.navigate()` without knowing the internal decision-making process.

Polymorphism

Different navigation strategies can be implemented and switched at runtime while maintaining the same interface for movement.

Inheritance

The architecture supports inheritance for creating specialized robots (e.g., SmartRobot, SwarmRobot) with extended capabilities.

import random
import time

# ================ GRID ENVIRONMENT CLASS ================
class Grid:
    def __init__(self, size, obstacles):
        self.size = size        # Encapsulation: size is private
        self.obstacles = obstacles

    def is_obstacle(self, x, y):
        # Check boundaries and obstacles
        return (x, y) in self.obstacles or x < 0 or y < 0 or x >= self.size or y >= self.size

    def display(self, robot):
        # Display grid with robot and obstacles
        for y in range(self.size - 1, -1, -1):
            for x in range(self.size):
                if robot.x == x and robot.y == y:
                    print(" R ", end="")
                elif (x, y) in self.obstacles:
                    print(" X ", end="")
                else:
                    print(" . ", end="")
            print()
        print("-" * 20)


# ================ SENSOR CLASS ================
class ObstacleSensor:
    def detect(self, grid, x, y):
        # Abstraction: simple interface for detection
        return grid.is_obstacle(x, y)


# ================ ROBOT CLASS ================
class Robot:
    def __init__(self, x, y):
        self.x = x              # Encapsulated position
        self.y = y
        self.direction = random.choice(["up", "down", "left", "right"])

    def next_position(self):
        # Polymorphism: can be overridden for different movement strategies
        if self.direction == "up":
            return self.x, self.y + 1
        elif self.direction == "down":
            return self.x, self.y - 1
        elif self.direction == "left":
            return self.x - 1, self.y
        else:
            return self.x + 1, self.y

    def move(self):
        nx, ny = self.next_position()
        self.x, self.y = nx, ny

    def change_direction(self):
        self.direction = random.choice(["up", "down", "left", "right"])


# ================ CONTROLLER CLASS ================
class Controller:
    def __init__(self, robot, sensor, grid):
        self.robot = robot
        self.sensor = sensor
        self.grid = grid

    def navigate(self):
        # Strategy Pattern: decision-making logic
        nx, ny = self.robot.next_position()
        if self.sensor.detect(self.grid, nx, ny):
            print("⚠ Obstacle detected → Changing direction")
            self.robot.change_direction()
        else:
            self.robot.move()
            print(f"✅ Robot moved {self.robot.direction}")


# ================ MAIN PROGRAM ================
# Object Creation and Interaction
grid = Grid(5, obstacles=[(1, 1), (2, 2), (3, 1), (3, 3), (4, 0)])
robot = Robot(0, 0)
sensor = ObstacleSensor()
controller = Controller(robot, sensor, grid)

# Simulation Loop (20 steps)
for step in range(20):
    print(f"\nStep {step + 1}")
    controller.navigate()
    grid.display(robot)
    time.sleep(1)
